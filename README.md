# 🚇 Seoul Subway Ridership Data Analysis

서울시 지하철 **승·하차 인원 공공데이터**를 활용하여  
지하철 이용 패턴을 분석하고, 데이터 전처리부터 시각화까지의  
**데이터 분석 전체 흐름을 실습한 프로젝트**입니다.

---

## 📌 Project Overview
- **Goal**  
  공공 교통 데이터를 기반으로 지하철 이용 패턴을 분석하고,  
  데이터 분석 프로세스(전처리 → 분석 → 시각화)를 체계적으로 정리

- **Key Focus**
  - 대규모 정형 데이터 처리
  - pandas 기반 데이터 전처리 및 그룹화
  - 분석 결과의 시각적 전달
